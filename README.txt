Il file jar è stato esportato come Runnable JAR file con la classe main Run. Di conseguenza eseguendolo si avrà l'esecuzione del Run. Qualora si volesse eseguire il PreRun occorre cambiare comando da linea di codice.

Eseguire Run : java -jar Prit2_MQS_Vers2_Pascale.jar
Eseguire PreRun : java -cp Prit2_MQS_Vers2_Pascale.jar Prit2_MQS_Vers2_Pascale.PreRun
